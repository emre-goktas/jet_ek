Gemini API Anahtarı

Akıllı isimlendirme için kendi Gemini API anahtarınızı kullanırsınız. Anahtar yalnızca bu tarayıcıda saklanır; sunucuya veya veritabanına asla yazılmaz.

Dikkat

  • Bu anahtar Google hesabınıza değil, bu tarayıcıya kayıtlıdır.
  • Bu bilgisayarı başka biri kullanırsa — kendi Gmail hesabıyla giriş yapsa bile — işlemler yine sizin anahtarınızla yapılır ve kullanım size fatura edilir.
  • Ortak/paylaşımlı bilgisayarlarda işiniz bitince "Anahtarı sil" ile temizleyin.
  • Farklı bir tarayıcı veya cihazdan girdiğinizde anahtarı yeniden girmeniz gerekir — otomatik senkronize olmaz.

API Anahtarı Nasıl Alınır?

  1. Adım 1: aistudio.google.com adresine gidin.
  2. Adım 2: Açılan sayfada Gmail hesabınızla giriş yapın.
  3. Adım 3: Sol sütunda bulunan anahtar simgesine ("Get API key") tıklayın.
  4. Adım 4: Sayfanın sağ üstünde bulunan "Create API key" butonuna tıklayın.
  5. Adım 5: Açılan pencerede "Choose an imported project" yazısının altındaki "+ Create project" seçeneğine tıklayın.
  6. Adım 6: Projeye istediğiniz bir isim verin ya da varsayılan ismi kabul edip "Create project" butonuna tıklayın.
  7. Adım 7: Açılan pencerede "Create key" butonuna tıklayın.
  8. Adım 8: "API key details" adında yeni bir pencere açılacak. "API Key" satırındaki kopyalama simgesine tıklayın.
  9. Adım 9: JETEK'teki ilgili kutucuğa yapıştırın ve "Kaydet" butonuna tıklayın.

Son Kontrol — Ücretsiz Olduğundan Emin Olun

Google AI Studio'da sol sütundaki ayarlar simgesine tıklayın; "Account status" bölümünde "Unpaid services" yazdığından emin olun. Aynı sütundaki "API keys" sayfasında "Set up billing" kısmında "Free tier" yazdığından emin olun. Bu kontrol ücretsiz sürümde olduğunuzu doğrular; kredi kartınızdan ücret kesilmez. (JETEK'teki "Adlandır" butonunun limiti ve hızı tamamen kullandığınız API anahtarıyla ilgilidir — ücretli anahtarlarda limit ve token sınırları çok daha esnektir.)
